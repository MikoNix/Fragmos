# Настройки разметки bock.py

Все настройки находятся в начале файла `bock.py` в секции **НАСТРОЙКИ РАЗМЕТКИ** (~строка 65).  
Все значения — пиксели Draw.io.

---

## Таблица констант

| Константа | Значение по умолчанию | Что регулирует |
|---|---|---|
| `_BLOCK_W` | `200` | Ширина любого блока (процесс, ввод/вывод, ромб, терминатор) |
| `_BLOCK_H` | `60` | Высота любого блока |
| `_CENTER_X` | `300` | X-координата центра главной линии |
| `_START_Y` | `60` | Y-координата первого блока («Начало») |
| `_STEP_Y` | `100` | Расстояние между блоками на **главной линии** |
| `_BRANCH_STEP_Y` | `80` | Расстояние между блоками **внутри ветки** if/else |
| `_BRANCH_X` | `220` | Горизонтальный отступ веток от центра |
| `_LABEL_W` | `40` | Ширина меток «ДА» / «НЕТ» |
| `_LABEL_H` | `20` | Высота меток «ДА» / «НЕТ» |
| `_LABEL_OFFSET` | `25` | Отступ метки от края ромба по горизонтали |
| `_LABEL_FONT` | `11` | Размер шрифта меток «ДА» / «НЕТ» |

---

## Схема расположения

```
        _CENTER_X (300)
             │
     ┌───────┴───────┐   ← _BLOCK_W (200)
     │    Начало     │   ← высота _BLOCK_H (60)
     └───────┬───────┘   ← _START_Y (60)
             │ ↕ _STEP_Y (100)
     ┌───────┴───────┐
     │   x = a + b   │
     └───────┬───────┘
             │ ↕ _STEP_Y (100)
     ┌───────┴───────┐
  НЕТ◄   условие   ►ДА    ← метки на расстоянии _LABEL_OFFSET (25) от края ромба
     └───────────────┘
     │               │
     │ ↕ _BRANCH_    │ ↕ _BRANCH_
     │   STEP_Y (80) │   STEP_Y (80)
  ┌──┴──┐         ┌──┴──┐
  │ ... │         │ ... │
  └──┬──┘         └──┬──┘
     │               │
     └───────┬────────┘  ← точка слияния (join)
             │
          _CENTER_X
     ┌───────┴───────┐
     │    Конец      │
     └───────────────┘


  no_x = _CENTER_X - _BRANCH_X (300 - 220 = 80)
  yes_x = _CENTER_X + _BRANCH_X (300 + 220 = 520)
```

## Частые правки

**Ветки if слишком далеко друг от друга по горизонтали:**
```python
_BRANCH_X = 160   # было 220
```

**Ветки if слишком далеко по вертикали:**
```python
_BRANCH_STEP_Y = 60   # было 80
```

**Вся схема слишком растянута по вертикали:**
```python
_STEP_Y = 80   # было 100
```

**Блоки слишком узкие — текст не влезает:**
```python
_BLOCK_W = 240   # было 200
```

**Сдвинуть всю схему правее (больше места слева для ветки НЕТ):**
```python
_CENTER_X = 400   # было 300
```
